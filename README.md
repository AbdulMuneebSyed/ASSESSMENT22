## How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd assesment
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open the app in your browser at `http://localhost:3000`.
